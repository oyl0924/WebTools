# WebTools 项目文件说明

## 项目概述
WebTools 是一个基于 Electron + Vue 3 的桌面应用，用于快速管理和访问常用网站，支持创建桌面快捷方式和多标签页功能。

## 目录结构说明

### 根目录文件

#### 配置文件
- **`package.json`** - 项目配置文件，包含依赖、脚本和构建配置
- **`vite.config.ts`** - Vite 构建工具配置，集成 Electron 插件
- **`tsconfig.json`** - TypeScript 编译配置
- **`tsconfig.node.json`** - Node.js 环境的 TypeScript 配置
- **`electron-builder.json5`** - Electron 应用打包配置（模板文件）
- **`.gitignore`** - Git 忽略文件配置

#### 文档和说明
- **`README.md`** - 项目详细说明文档，包含功能介绍、使用指南等
- **`CLAUDE.md`** - Claude Code 项目规范说明
- **`yarn.lock`** - Yarn 依赖锁定文件
- **`index.html`** - 应用主 HTML 文件

### 主要目录

#### `electron/` - Electron 主进程代码
- **`main.ts`** - 主进程入口文件，负责窗口管理、IPC 通信等
- **`preload.ts`** - 预加载脚本，提供渲染进程与主进程的安全通信
- **`storage.ts`** - 本地数据存储服务，管理网站数据持久化
- **`electron-env.d.ts`** - Electron 环境类型定义
- **`childWindowTemplate.ts`** - 子窗口 HTML 模板生成器
- **`services/settingsService.ts`** - 设置服务

#### `src/` - Vue 前端代码
- **`main.ts`** - Vue 应用入口文件，配置 Ant Design Vue、路由等
- **`App.vue`** - 根组件，基础样式配置
- **`router.ts`** - Vue Router 路由配置
- **`style.css`** - 全局样式文件
- **`vite-env.d.ts`** - Vite 环境类型定义

#### `src/components/` - Vue 组件
- **`HomePage.vue`** - 主页面组件，网站列表展示
- **`ManageModal.vue`** - 网站管理弹窗组件
- **`AddWebsiteModal.vue`** - 添加网站弹窗组件
- **`EditWebsiteModal.vue`** - 编辑网站弹窗组件
- **`CustomButtonModal.vue`** - 自定义按钮管理弹窗组件
- **`SettingsModal.vue`** - 设置弹窗组件
- **`WebViewWindow.vue`** - WebView 窗口组件
- **`HelloWorld.vue`** - 示例组件（未使用）

#### `src/types/` - TypeScript 类型定义
- **`index.ts`** - 项目类型定义文件

#### `src/styles/` - 样式文件
- **`dark-theme.css`** - 深色主题样式

#### `src/utils/` - 工具函数
- **`index.ts`** - 工具函数集合

#### `src/assets/` - 静态资源
- 存放图片、图标等资源文件

#### `public/` - 公共静态资源
- **`icon.ico`** - 应用图标
- **`title-bar.css`** - 标题栏样式文件
- **`vite.svg`** - Vite 图标
- **`electron-vite.svg`** - Electron-Vite 图标
- **`electron-vite.animate.svg`** - 动画版 Electron-Vite 图标

#### `dist/` - 构建输出目录
- 存放 Vite 构建后的前端代码
- `index.html` - 构建后的主页面
- `assets/` - 构建后的静态资源

#### `dist-electron/` - Electron 构建输出目录
- **`main.js`** - 编译后的主进程代码
- **`preload.mjs`** - 编译后的预加载脚本

#### `release/` - 应用打包输出目录
- 存放 electron-builder 打包后的安装程序

#### `node_modules/` - 依赖包目录
- 存放项目依赖的 npm 包

#### `.vscode/` - VS Code 配置
- **`extensions.json`** - 推荐的扩展插件配置

#### `.git/` - Git 版本控制目录
- Git 仓库相关文件

## 文件作用详细说明

### 核心功能文件

1. **`electron/main.ts`** - 应用核心逻辑
   - 窗口创建和管理
   - IPC 通信处理
   - 多标签页支持
   - 桌面快捷方式创建
   - WebView 生命周期管理

2. **`electron/storage.ts`** - 数据持久化
   - 网站数据存储和读取
   - 自定义按钮管理
   - JSON 文件操作

3. **`src/components/HomePage.vue`** - 主界面
   - 网站卡片网格展示
   - 搜索功能
   - 网站打开和管理

4. **`src/components/ManageModal.vue`** - 管理界面
   - 网站列表管理
   - 添加/编辑/删除网站
   - 自定义按钮配置
   - 桌面快捷方式创建

### 配置文件说明

1. **`vite.config.ts`** - 构建配置
   - Vue 插件配置
   - Electron 插件集成
   - 开发服务器配置

2. **`package.json`** - 项目元数据
   - 依赖包管理
   - 构建脚本定义
   - Electron 构建配置

3. **`electron-builder.json5`** - 打包配置模板
   - 多平台打包配置
   - NSIS 安装程序配置
   - 文件包含规则

## 建议删除的文件

### 可安全删除的文件

1. **`src/components/HelloWorld.vue`**
   - 原因：Vue 模板示例组件，项目中未使用
   - 影响：无影响

2. **`public/vite.svg`**
   - 原因：Vite 框架示例图标，项目中未引用
   - 影响：无影响

3. **`public/electron-vite.svg`** 和 **`public/electron-vite.animate.svg`**
   - 原因：Electron-Vite 示例图标，项目中未引用
   - 影响：无影响

4. **`electron-builder.json5`**（可选）
   - 原因：打包配置模板，实际配置已在 `package.json` 中
   - 影响：如果不需要模板文件可删除

5. **空的组件目录**
   - `src/components/HomePage/` - 空目录
   - `src/components/BookmarkEditor/` - 空目录
   - `src/components/WebsiteIcon/` - 空目录
   - 原因：这些目录没有实际文件
   - 影响：无影响

### 注意事项

在删除文件前，请确保：
1. 文件确实没有被其他代码引用
2. 已备份重要文件
3. 删除后重新构建项目确认无错误

## 构建和运行

### 开发模式
```bash
yarn dev
```

### 构建 Windows 版本
```bash
yarn build:win
```

### 构建所有平台
```bash
yarn build
```

构建完成后，安装包位于 `release` 目录下。
